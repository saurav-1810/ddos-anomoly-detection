Step-by-Step Instructions:

1. Download and install PyCharm and Python 3.12 (latest version) from Google Chrome.
2. Open PyCharm and create a new Python environment for your project.
3. In the terminal, run the following command to install the required dependencies from the requirements.txt file:
4. pip install -r requirements.txt
5. Run the app.py file to start the application.
6. Copy the local URL (e.g., http://127.0.0.1:5000/) displayed in the terminal and paste it into Chrome.
7. Use the web interface to upload the test file.
8. Complete the process.
